# Інформація про SEO налаштування

## Де знаходиться SEO код:

### 1. **index.html** (основні мета-теги)
Файл: `index.html` (в корені проекту)

Містить базові SEO теги:
- `<title>` - заголовок сторінки
- `<meta name="description">` - опис сторінки
- `<meta name="keywords">` - ключові слова
- `<meta name="author">` - автор
- `<meta name="robots">` - інструкції для пошукових роботів
- `<meta property="og:...">` - Open Graph теги для соцмереж
- `<link rel="canonical">` - канонічний URL

### 2. **App.jsx** (динамічні мета-теги через react-helmet)
Файл: `src/App.jsx`

Використовує компонент `<Helmet>` з бібліотеки `react-helmet` для додавання мета-тегів динамічно.

## Поточні SEO налаштування:

### Title (Заголовок):
```
Vertex - Діджитал Агенція | Створення Сайтів та Веб-Розробка в Україні
```

### Description (Опис):
```
Vertex - професійна діджитал агенція. Створюємо веб-сайти, односторінкові сайти, інтернет-магазини та корпоративні сайти. Гарантія 5 років, підтримка 24/7. Від 300$. Харків, Україна.
```

### Keywords (Ключові слова):
```
створення сайтів, веб-розробка, діджитал агенція, односторінковий сайт, інтернет-магазин, корпоративний сайт, веб-дизайн, розробка сайтів Україна, Харків, Vertex
```

### Open Graph (для соцмереж):
- og:title: "Vertex - Діджитал Агенція | Створення Сайтів"
- og:description: "Професійна розробка сайтів та веб-рішень. Гарантія 5 років, підтримка 24/7. Від 300$."
- og:type: "website"

### Canonical URL:
```
https://vertex-web.com.ua
```

## Як змінити SEO:

1. **Для зміни базових тегів:** Відредагуйте `index.html`
2. **Для зміни динамічних тегів:** Відредагуйте `<Helmet>` компонент в `src/App.jsx`

## Рекомендації:

- Title має бути 50-60 символів
- Description має бути 150-160 символів
- Keywords можна розширити додавши більше релевантних слів
- Canonical URL змініть на ваш реальний домен
